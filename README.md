# Customer Data Preprocessing & Analysis

This project was completed as part of the MSc Artificial Intelligence and Data Science programme.

## Project Overview
The project focuses on preparing raw customer data exported in CSV format for data science analysis.
It includes data cleaning, restructuring, validation, and feature engineering.

## Key Tasks
- Read and process large CSV datasets using Python
- Convert flat data into nested JSON structures
- Handle missing and inconsistent data values
- Perform feature engineering (salary-to-commute ratio)
- Filter datasets by employment and retirement status
- Detect invalid credit card records
- Perform exploratory data analysis and visualisation

## Technologies Used
- Python
- CSV, JSON
- Pandas
- Seaborn
- Jupyter Notebook

## Outputs
- Cleaned and structured JSON files
- Visual insights into customer data
